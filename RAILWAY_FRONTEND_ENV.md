# Переменные окружения для Railway Frontend

Установите следующие переменные окружения в Railway Dashboard для фронтенда:

## Обязательные переменные:

```
NEXT_PUBLIC_API_URL=https://fullstack-lab-work-123.up.railway.app
```

## Инструкция:

1. Откройте ваш **Frontend Service** в Railway Dashboard
2. Перейдите в раздел **Variables**
3. Добавьте переменную:
   - **Key:** `NEXT_PUBLIC_API_URL`
   - **Value:** `https://fullstack-lab-work-123.up.railway.app`
4. Сохраните изменения
5. Railway автоматически перезапустит сервис

## Проверка:

После перезапуска фронтенд должен подключиться к бэкенду:
- Фронтенд: https://web-123-b09c.up.railway.app
- Бэкенд: https://fullstack-lab-work-123.up.railway.app
